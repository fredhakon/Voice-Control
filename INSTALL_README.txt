VOICE CONTROL PLATFORM - Installation Guide
============================================

Thank you for installing Voice Control Platform!

REQUIREMENTS:
- Windows 10 or later
- Microphone (for voice commands)
- Speakers (for audio feedback)

FEATURES:
- Voice-activated commands for controlling your PC
- System controls (volume, media playback)
- Custom commands to open files and applications
- Keyboard macro recording and playback
- Customizable voice phrases
- Global keyboard shortcut to toggle listening

FIRST TIME SETUP:
1. After installation, launch Voice Control Platform
2. Go to Settings to select your microphone
3. Click "Calibrate Microphone" in a quiet environment
4. Optionally set up a custom keyboard shortcut

DEFAULT COMMANDS:
- "Volume up" / "Volume down" - Adjust system volume
- "Mute" / "Unmute" - Toggle mute
- "Play" / "Pause" / "Next" / "Previous" - Media controls

ADDING CUSTOM COMMANDS:
1. Click "Add Command" in the main window
2. Enter voice phrases (one per line)
3. Choose to open a file/app or record a keyboard macro
4. Save and start using your new command!

KEYBOARD SHORTCUT:
- Default: Ctrl+Shift+V to toggle listening on/off
- Can be changed in Settings

SUPPORT:
For help and updates, visit the project repository.

Enjoy hands-free control of your PC!
